With this Widget you can include your PHP Code in a Widget.

== Description ==

This Widget is like a raw Widget. You can put every PHP Code into the Widget. It gives you the possibility to make non-widget features with a couple of clicks to an Widget.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `widgetizeit.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the Widget where you like

== Example ==

You have a nice Plugin but it do not fit into you Widget Design. But to write a widget is to time-consuming. So go to your adminpanel, add a new Instance of Widgetize It and include the code. Finish!
